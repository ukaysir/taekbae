# HWP 작성 상태

`full_draft.json`은 붙임 3 원본 양식에 제안서·분석과정 보고서 초안을 넣기 위해 만든 **계획 산출물**이다. 완성 HWP가 아니며 현재 상태로 자동 실행해 제출해서는 안 된다.

2026-07-15 안전 편집 도구로 원본 해시 확인, 구조 분석, 6쪽 렌더링 검수와 44개 치환 작업의 사전 계획까지 완료했다. 상가정보 행과 검증 수치도 계획에 반영했다. 실제 편집 단계에서는 다음 두 조건 때문에 도구가 결과 파일을 폐기하고 중단했다.

- 일부 표 셀의 첫 문단에 인라인 개체가 있어 일괄 치환 시 개체가 유실될 수 있다.
- 장문 셀은 여러 문단으로 구성되어 있어 첫 문단만 바꾸면 원래 안내문이 남고, 자동 의미 검증을 통과하지 못한다.

HWP 내부 바이트나 XML을 직접 고치는 우회 방식은 원본 훼손 위험 때문에 사용하지 않았다. 따라서 현재 제출용으로 생성된 HWP는 없다. 한컴오피스에서 원본 양식을 직접 열어 PDF 초안의 내용을 옮기고, 아래 사용자 입력을 받은 뒤 육안 검수해야 한다.

- 개인/팀 인적사항과 연락처
- 개인정보 수집·이용 및 저작물 이용 동의 선택
- 서명 또는 날인
- 심사위원 기피 신청 번호 7개

PDF 초안은 `outputs/submission/`에 별도로 생성되며, `[DRAFT]` 표기와 `[사용자 입력 필요]` 자리표시자를 유지한다.
