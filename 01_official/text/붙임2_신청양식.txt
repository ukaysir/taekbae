   ○ 분석 과정에서 AI를 어떻게 활용했는지 도구명과 활용목적 기재(하단 예시 참고)
    - (예시) ChatGPT-4o : 데이터 전처리 코드 생성 보조 / Claude : 분석 방향 검토
    - (활용목적 예시) 아이디어 구체화, 데이터 전처리 코드 생성, 분석 방법론 검토, 결과 해석 보조, 문서 초안 작성 등
 3. 주요 프롬프트(3개 이상, 각 3~5줄 작성)
   ○ 분석 과정에서 사용한 핵심 프롬프트를 3개 이상 기재(하단 예시 참고)
     - 활용 단계: 데이터 전처리 코드 생성
     - 입력 프롬프트: "pandas를 사용하여 연도별·월별 물량 데이터에서 이상치를 탐지하고 제거하는 파이썬 코드를 작성해주세요“
     - AI 출력 결과 요약: IQR 방식의 이상치 탐지 코드 제공
 4. AI 결과물 검토 및 수정 과정(1~2개 사례)
   ○ AI가 생성한 결과물 중 수정하거나 보완한 내용과 그 이유 기술(하단 예시 참고)
     - AI 생성 결과(문제점): 물량 예측 모델에서 공휴일 변수를 반영하지 않아 명절 주간 예측 오차가 큼
     - 수정·보완 내용: 공휴일 더미 변수를 직접 추가하고 모델 재학습 수행
     - 수정 이유: 국내 물류 물량은 명절에 크게 영향을 받으므로 해당 변수 반영이 필수적이라 판단
 5. 팀 기여 내용(3~5줄 작성)
   ○ AI 활용 전후 본인(팀)이 직접 수행한 분석·판단·해석 내용(하단 예시 참고)
     - 문제 정의 및 분석 방향 설정: 팀 전원 직접 수행
     - 데이터 수집 및 정제: 팀원 A, B 직접 수행 (AI는 코드 보조에만 활용)
     - 분석 결과 해석 및 인사이트 도출: 팀 전원 직접 수행
     - 최종 제안서 작성: AI 초안 활용 후 팀 전원이 내용 수정·보완
