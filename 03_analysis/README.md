# 분석 근거

이 폴더에는 제안서와 분석 과정 보고서에 실제로 사용할 근거만 남겼다.

## 검증된 핵심 결과

- 교통관측 품질: 11스냅숏·451행·41구간
- 상가정보: 대전 78,607행 좌표 유효
- 배송노출 대리값: 계족로 69곳, 읍내삼거리 12곳, 계백로 994곳
- 공사범위 공식 근거 3건 검증
- 서대전 양방향 표준 통행축 8개 링크 재현

## 폴더

- `data/`: 사람이 검수한 공사 이벤트·매핑·공간근거 CSV
- `results/`: 품질·매핑·노출 검증 JSON/CSV
- `references/`: 출처대장, 조사기록, API 검증, 경쟁서비스 비교

`current_risk.csv`는 고정 관측자료의 표 형식 예시다. 예측값이나 실제 배송성과로 해석하지 않는다.

이전에 만든 대시보드 시안은 현재 고정 수치와 달라 2026-07-20 폐기했다. 새 시각 디자인을 만들기 전까지 화면요소와 표시 데이터는 `02_submission/ui_ux_data_spec.md`에서만 관리한다.
