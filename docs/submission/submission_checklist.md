## 주최기관 확인 필요

- [ ] 공고문은 저작재산권이 원칙적으로 참가자에게 귀속된다고 하나 서식 4 참가서약서 제4항은 주최기관 이양으로 되어 있어, 어느 조항이 적용되는지 서면 확인
- [ ] 참가신청서의 최종 제출 형식(HWP 또는 PDF) 확인
- [ ] 심사위원 추첨번호는 공고문의 `○ 표시`와 서식 5의 `빨간색 폰트`를 함께 적용하면 되는지 확인

## 서식·파일

- [ ] 서식 1 참가신청서 완성
- [ ] 서식 2 제안서 HWP 10쪽 이내, 쪽번호
- [ ] 서식 2 제안서 PDF 10쪽 이내, 글자·표·URL 확인
- [ ] 서식 3 분석 과정 보고서 HWP, 3~5쪽 권장
- [ ] 서식 3 분석 과정 보고서 PDF, 그림 해상도 확인
- [ ] 서식 4 개인정보 동의·참가서약 PDF, 팀원 전원 서명
- [ ] 서식 5 심사위원 추첨표 PDF, 정확히 7개 번호를 빨간색으로 표시하고 ○ 처리, 전원 서명
- [ ] 참가신청서 파일 형식이 PMS 매뉴얼 요구와 일치
- [ ] 모든 파일명에 팀명 반영
- [ ] ZIP 내부에 비밀값·원자료 개인정보·불필요한 개발파일 없음
- [ ] ZIP 50MB 이하
