# 참가자 입력 템플릿

개인정보가 포함된 작성본은 Git에 커밋하지 않는다. 이 템플릿을 복사해 저장소 밖 또는 `.private/` 폴더에 작성하고, 최종 HWP 입력 후 안전하게 보관한다.

## 서식 1

- 참가유형: 개인 / 팀
- 신분유형: 학생 / 직장인 / 일반 / 기타
- 팀명: 개인 출품도 필요, 개인 실명 지양
- 주소: 광역시도·시군구까지만
- 팀장: 이름 / 소속 / 생년월일 / 휴대전화 / 이메일
- 팀원 1~4: 이름 / 소속 / 생년월일 / 휴대전화 / 이메일
- 미성년 팀 보호자 정보: 해당 시에만
- 참가경로: 인터넷 URL / 포스터 / SNS / 지인 추천 / 기타
- 가점 해당사항: 공식 서식의 해당 선택지를 본인이 확인
- 신청일과 대표자 서명 또는 날인

## 서식 4

팀원별로 각각 확인한다.

- 개인정보 수집·이용: 동의 / 미동의
- 홍보 목적 이용: 동의 / 미동의
- 참가서약: 동의 / 미동의
- 작성일
- 팀장과 팀원 전원 서명

저작재산권 조항은 공고문과 참가서약서가 충돌하므로 주최기관 답변 또는 정정서식 확인 후 서명한다.

## 서식 5

- 1~35 중 서로 다른 번호 정확히 7개
- 빨간색 폰트와 ○ 표시 방법은 주최기관 답변 확인
- 작성일
- 팀장과 팀원 전원 서명

## PMS

- 팀장 대표계정과 본인인증
- PMS에 입력할 팀명과 지원과제명
- 공식 지원과제명: `대전 트램 공사구간의 도심배송 위험 관측·예보 모듈`
