# 제출용 시각자료

제안서와 분석 과정 보고서에 삽입할 16:9 시각자료의 원본 HTML/CSS와 PNG를 관리한다.

## 구성

- `src/01_service_flow.html`: 데이터·AI·사람의 역할과 서비스 흐름
- `src/02_verification_workspace.html`: 읍내삼거리 공사정보·도로구간 검증 화면
- `src/03_evidence_summary.html`: 고정 데이터와 분석결과 요약
- `png/`: 위 HTML을 1600×900으로 캡처한 제출용 이미지

## 표현 원칙

- 실제로 확인한 수치만 사용한다.
- 화면은 개념 시안으로 표시하며 운영 중인 서비스로 표현하지 않는다.
- 451개 교통기록은 데이터 결합 가능성 확인용이며 예측 성능으로 제시하지 않는다.
- 반경 250m 상가 수는 택배 물량이 아니라 배송처 분포의 간접지표다.
- AI가 최종 도로구간·배차·경로를 결정하는 것으로 표현하지 않는다.
