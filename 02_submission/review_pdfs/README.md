# 제출 초안 안내

이 폴더의 `[DRAFT]` PDF 2개는 내용·페이지 배치 검토용이며 최종 제출본이 아니다. 2026-07-20 수집 종료 결정은 Markdown 초안에 반영됐지만 새 clone에 `.venv`가 없어 PDF를 아직 재생성하지 못했으므로, 현재 PDF 내용은 이전 판본이다.

- `[DRAFT]제안서.pdf`: A4 8쪽, 공고상 10쪽 이내 조건 자동검사 완료
- `[DRAFT]분석과정보고서.pdf`: A4 3쪽, 권장 3~5쪽 조건 자동검사 완료

두 파일 모두 텍스트 추출, 페이지 크기, 빈 페이지 여부와 렌더링을 검사했고 Edge 기본 머리말·꼬리말 및 로컬 파일 경로를 제거했다. 다만 `[사용자 입력 필요]` 자리표시자와 `DRAFT · 검토용` 워터마크가 남아 있으므로 그대로 제출하면 안 된다.

최종 제출 전에는 다음 작업이 필요하다.

1. `02_submission/submission_checklist.md`의 사용자 입력과 데이터 마감 항목을 완료한다.
2. 공식 HWP 양식에 내용을 직접 옮기고 표·쪽번호·서명을 육안 검수한다.
3. 팀명·개인정보·동의·서명·심사위원 번호를 반영한다.
4. 추가 수집은 하지 않는다. `03_analysis/results/traffic_quality.json`의 11스냅숏·451행과 공간매핑·상가노출 결과를 고정해 반영하고 30분 예측 성능은 주장하지 않는다.
5. `[DRAFT]` 표시를 제거한 필수 서류 5종의 실제 파일 7개를 `02_submission/required_documents.md`에 적힌 이름과 형식으로 준비한다.
6. `02_submission/tools/verify_submission.py`로 페이지 수와 자리표시자를 확인하고, 최종 ZIP은 50MB 미만인지와 압축 해제 여부를 직접 점검한다.

HWP는 `01_official/originals/[붙임2] 신청양식.hwp` 원본을 한컴오피스에서 직접 작성하고 육안 검수한다.
