# 제출 준비

## 핵심 문서

- `required_documents.md`: 제출할 서류 5종·실제 파일 7개
- `submission_checklist.md`: 제출 전 확인사항
- `plan.md`: 수집 종료가 반영된 실행계획
- `ui_ux_data_spec.md`: 디자인 시스템 적용 전에 확정한 필수 화면·구성요소·표시 데이터
- `drafts/`: 제안서와 분석 과정 보고서의 최신 Markdown 초안
- `review_pdfs/`: 최신 Markdown 초안으로 생성한 검토용 PDF
- `participant_input_template.md`: 참가자가 직접 입력해야 할 정보
- `organizer_question_draft.md`: 권리조항·추첨표기 확인 문의문

공식 HWP는 `01_official/originals/[붙임2] 신청양식.hwp`를 직접 열어 수동으로 작성한다. 검토용 PDF는 제출본이 아니다.

기존 대시보드 시안은 확정 수치와 충돌해 폐기했다. 새 화면 이미지는 사용자가 제공할 디자인 시스템을 받은 뒤 `ui_ux_data_spec.md`의 요소와 데이터만 사용해 만든다.
