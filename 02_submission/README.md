# 제출 준비

## 핵심 문서

- `required_documents.md`: 제출할 서류 5종·실제 파일 7개
- `submission_checklist.md`: 제출 전 확인사항
- `plan.md`: 수집 종료가 반영된 실행계획
- `ui_ux_data_spec.md`: 디자인 시스템 적용 전에 확정한 필수 화면·구성요소·표시 데이터
- `drafts/`: 제안서와 분석 과정 보고서의 본문 원고
- `visuals/`: HTML/CSS 원본과 1600×900 제출용 PNG 3종
- `manuscripts/`: 이미지가 포함된 HWP·PDF 원고와 주의사항
- `participant_input_template.md`: 참가자가 직접 입력해야 할 정보
- `organizer_question_draft.md`: 권리조항·추첨표기 확인 문의문

`manuscripts/`의 제안서 10쪽·분석 과정 보고서 4쪽 HWP/PDF는 같은 한컴 문서에서 생성했고 페이지·본문·이미지 포함 여부를 검사했다. 팀명·개인정보·동의·서명은 사실확인이 필요하므로 자동 작성하지 않았으며, 이를 입력하기 전에는 최종 제출본이 아니다.

기존 대시보드 시안은 확정 수치와 충돌해 폐기했다. 새 이미지는 제공된 레퍼런스의 디자인 언어를 재해석하되 `ui_ux_data_spec.md`의 확정 요소와 고정 분석값만 사용했다.
