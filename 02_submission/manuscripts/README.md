# 제출 원고 파일

제안서와 분석 과정 보고서의 본문·표·시각자료를 포함한 HWP/PDF 원고를 둔다.

- `2. 제안서.hwp`, `2. 제안서.pdf`
- `3. 분석 과정 보고서.hwp`, `3. 분석 과정 보고서.pdf`

본문은 완성됐지만 팀명·대표자·팀원 개인정보와 서명은 사실확인 없이 만들 수 없어 `[사용자 입력 필요]`로 남겨 두었다. 해당 정보를 공식 양식 전체에 반영하고 최종 서명하기 전에는 제출본으로 사용하지 않는다.

HWP는 Markdown 원고의 문단·표·이미지를 한컴 문서 객체로 직접 구성해 저장한다. PDF는 같은 한컴 문서에서 생성하므로 HWP와 내용이 같다. 외부 문서 변환이나 원본 HWP 바이너리 수정은 사용하지 않는다.
